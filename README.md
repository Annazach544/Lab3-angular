# Angular Lab - Enhetskonverterarr
Detta projekt är en webbapplikation byggd med Angular som en del av en labbuppgift.

## Syfte 
Syftet med uppgiften är att få grundläggande förståelse för Angular som ramverk, inklusive: 
- komponenter
- routing
- databindning
- styling

## Funktioner 
Webbplatsen innehåller: 
- Startsida med information och bild
- About-sida med beskrivning av arbetet
- Converter-sida för enhetsomvandling

## Konverterare
På konverteringssidan kan användaren: 
- konvertera meter ↔ feet
- konvertera Celsius ↔ Fahrenheit

## Tekniker som används
- Angular
- TypeScript
- HTML / CSS
- Angular Router
- Two-way data binding (ngModel)
- Components
